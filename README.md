# A fullstack project.

Prisma, Nuxt3 were used to create this project.
Plus few modules like TailwindCSS, PrimeVue

# Init 
```bash
npm install
```

# Run dev server
```bash
npm run dev
```

# Build
```bash
npm run build
```

# Misc

```bash
# Get local display of DB (ENV is needed)
npm run prisma:studio

# Set remote DB, based on your prisma scheme
npm run prisma:push
